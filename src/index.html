Olodoc

Olodoc, Php dili ile geliştirilen hızlı ve kolayca dokumentasyon belgeleri oluşturmanızı sağlayan belge oluiturma aracıdır. Olodoc'ın geliştirilmesinde aşağıdaki teknolojiler kullanılmıştır.

  • Bootstrap (5.1.3)
  • Prism Js
  • Php
  • Socialite Social Login
  • Mezzio Framework
  • Olobase Mezzio
  • Jwt Authentication
  • Composer
  • Redis

Özellikler

Olodoc aşağıdaki özellikleri destekler:

  • Mark Down (.md) Dosyaları ile Çalışır
  • Jwt Kimlik Doğrulama
  • Kod Vurgulama
  • Bootstrap 5 Template Support
  • Çoklu Dil Desteği
  • Alt Menü Desteği
  • Sayfa Çubukları ile Sayfada Gezinme
  • MYSQL Tam Metin Arama
  • Base64 Kodlu Görüntü Oluşturma
  • Kaynak Sınırlama (Gizli veya Açık Belgeler)
  • Sekme Desteği
  • Sosyal Oturum Açma Desteği (Google, Github)
  • Çapraz Alan Adı Oturum Açma Desteği
  • Php Mezzio ve Doğal Javascript ile Kodlanmıştır
